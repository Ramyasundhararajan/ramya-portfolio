Research

Anomaly Detection in Network Traffic using Deep Learning Techniques

Abstract

In the Tech Renaissance, detecting network traffic anomalies is vital for real-time security against sophisticated cyber threats. Traditional rule-based methods struggle with evolving patterns, high false positives, and unknown threats. This research explores advanced machine learning techniques like autoencoders, RNNs, CNNs, and GANs for adaptive anomaly detection, reducing false positives and enhancing accuracy. The proposed approach offers improved security, reduced cyberattack risks, and robust protection of sensitive data.

Highlights

  • GAN-based approach outperformed traditional methods.
  • Reduced false positives and improved detection precision.
  • Presented at ICCDA conference.

Future Enhancements

Future work involves using a virtual assistant to manage and mitigate anomalies detected by deep learning models. The assistant will respond in real-time to alerts, implementing corrective actions to neutralize threats automatically. This integration enhances network security by swiftly addressing anomalies and reducing their impact. The approach streamlines anomaly management and improves system resilience.
